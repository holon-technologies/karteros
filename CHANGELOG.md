# Changelog

All notable changes to Karteros are documented here.

## [0.1.0] - 2026-07-18

### Added

- Canonical public Karteros architecture specification.
- Experimental accepted-operation contracts with opaque application payloads.
- Atomic SQLite command, operation, projection, and transactional outbox
  persistence.
- Idempotent duplicate handling and changed-operation conflict detection.
- Explicit process crash-point test helpers.
- Two-crash recovery reference process with an idempotent receiver effect.
- Pinned Rust 1.97.1 workspace and GitHub Actions verification.
